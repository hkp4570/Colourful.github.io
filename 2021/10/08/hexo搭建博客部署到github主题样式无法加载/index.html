hexo搭建博客部署到github主题样式无法加载

发表于 2021-10-08 分类于 记录

问题

使用hexo+NexT主题搭建完成个人博客，部署到github之后，打开发现设置的主题并没有生效。本地打开是正常的。

解决

之前看到有文章说是需要修改主题文件夹（/theme/next/source/）下的某个文件，不过新版本的中已经修过过了。如果你是这种问题请查看。
当然我遇到的问题是配置url错误。请在_config.yml中设置url为你博客的域名。
